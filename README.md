# ⏱️ Segna Ore

App web per registrare le ore di lavoro presso i clienti e calcolare al volo il **costo della manodopera** (imponibile, IVA e totale).

Pensata per chi lavora in cantiere o a domicilio: arrivi e premi **Arrivo**, finisci e premi **Finito** — l'app calcola da sola tempo e costo.

## 🔗 Demo dal vivo
👉 **https://maximilianred.github.io/segna-ore/**

## ✨ Funzionalità
- ▶️ Cronometro **Arrivo / Finito** con un tocco
- 💶 Calcolo automatico di **imponibile, IVA e totale** (tariffa e IVA impostabili)
- ⏲️ **Arrotondamento** del tempo a piacere (es. a 15 minuti)
- 📋 **Storico** dei lavori con i totali (ore, imponibile, totale con IVA)
- 📤 **Esportazione in CSV** per Excel / contabilità
- 💾 Dati salvati sul dispositivo: funziona **offline**, senza account
- 📱 Si aggiunge alla **schermata Home** del telefono come un'app

## 🛠️ Tecnologie
- **HTML, CSS e JavaScript** puro (vanilla), in **un solo file** `index.html`
- `localStorage` del browser per salvare i dati sul dispositivo
- Nessuna libreria esterna, nessun processo di build

## 🚀 Come si usa
1. Apri la [demo dal vivo](https://maximilianred.github.io/segna-ore/)
2. (Da telefono) menu del browser → **"Aggiungi a schermata Home"**
3. Scrivi il nome del cliente, premi **Arrivo** all'inizio e **Finito** alla fine
